# Transporte 54

Sitio web estático para el proyecto de transporte, con páginas de inicio, servicios, galería, novedades y contacto.

## Demo Online
[Ver sitio en vivo]

## Descripción del proyecto

Este proyecto es una página web informativa y responsiva para una empresa de transporte. Incluye:

- Página principal (`index.html`) con la presentación general.
- Secciones de servicios, novedades, galería e información sobre la empresa.
- Navegación entre páginas secundarias en la carpeta `page/`.
- Estilos personalizados en la carpeta `css/`.
- Imágenes organizadas en `imagenes/`.
- Posible uso de JavaScript en la carpeta `js/`.

## Estructura de archivos

- `index.html` - Página principal del sitio.
- `css/` - Archivos de estilo CSS.
- `js/` - Archivos JavaScript (si se utilizan).
- `imagenes/` - Imágenes y galerías del sitio.
- `page/` - Páginas secundarias: `contactos.html`, `galeria.html`, `nosotros.html`, `novedades.html`, `servicio.html`.

## Tecnologías utilizadas

- **HTML**
- **CSS**
- **Bootstrap 5**
- JavaScript (opcional)

## Cómo usar

1. Abrir `index.html` en un navegador web.
2. Navegar por las secciones del sitio usando los enlaces del menú.

## Notas

Este proyecto está diseñado como una página web estática. No requiere servidor ni instalación especial, solo un navegador para abrir los archivos HTML.
